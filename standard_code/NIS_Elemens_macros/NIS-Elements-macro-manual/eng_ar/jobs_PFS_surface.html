Creating PFS Surface

Perfect Focus System Surface, displayed in the XYZ Overview (see: View > Acquisition Controls > XYZ Overview ) can be defined using a job. Following example shows PFS Surface definition on a well plate.

Figure 1081. 

PFS > Auto PFS Focus Setup task defines settings(Optical configuration) to be used by thePFS > Auto PFS Focus task. On each well (Well Plates > Loop over Wells) of the well selection (Well Plates > Select Wells), the stage moves to the center of a well (Well Plates > Move to Well Center), turns on PFS (PFS > PFS On and Focus), automatically finds the PFS offset value (PFS > Auto PFS Focus), assigns it to the each well (PFS > Assign PFS to Point/Well) and turns PFS off (PFS > PFS Off). Once all wells undergo this procedure, the PFS Surface is finally created on the well selection using the PFS > Create PFS Surface task.
