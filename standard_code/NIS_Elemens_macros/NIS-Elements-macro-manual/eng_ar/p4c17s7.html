Description 

 SMTPConfig(
   char *SMTPServerName,
   int  SMTPPort,
   int  SMTPUseDefalutSecurityType,
   int  SMTPSecurityType,
   char *SMTPLogin,
   char *SMTPPassword,
   char *SMTPSenderMail,
   char *SMTPSenderName
);

This function specifies server/mail settings which are used when the SMTPSendEmail function is called.

Parameters

char *SMTPServerName

SMTP server name. e.g. ‚Äúsmtp.mydomainname.com‚Äù or ‚Äúmail.mydomainname.com‚Äù

int SMTPPort

Port number

int SMTPUseDefalutSecurityType

Set security type based on the port number?

0

Use security type specified by the SMTPSecurityType parameter.

1

The SMTPSecurityType parameter is ignored. The security type is recognized according to SMTPPort (465:USE_SSL, 587:USE_TLS, 25 or any other:NO_SECURITY)

int SMTPSecurityType

Security protocol to be used for connection to the mailserver. Different protocols usually use different port numbers.

0

No encryption

1

TLS

2

SSL

char *SMTPLogin

If the SMTP server requires authentication, fill in the appropriate user account name e.g. ‚Äúfirstname.surname‚Äù.

char *SMTPPassword

If the SMTP server requires authentication, fill in the password for the SMTPLogin account

char *SMTPSenderMail

Email address which will be displayed in the ‚Äúfrom‚Äù field (and also used as the reply-to address)

char *SMTPSenderName

‚ÄúAlias‚Äù of the sender email - usually a human-readable equivalent of the email address. E.g. ‚ÄúPeter Smith‚Äù for an address such as ps001@company.com
