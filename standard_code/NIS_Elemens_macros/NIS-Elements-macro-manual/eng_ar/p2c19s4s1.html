Basic Workflow

  1. Make sure the Grain Size module is started ( Applications > Metallography > Grain Size)

  2. Click on the Grain Size Wizard button on the left tool bar.

  3. Select the measurement method, industrial standard and the image data source. The procedure then differs depending on the selected Image Data Source:

    Figure 1251. 

    Automatic Capture

    Requires a motorized stage and a camera. The user specifies size of the sample and the number of fields to be scanned.

    Manual Capture

    Requires a camera. The user is asked to capture a new image before each measurement.

    File Sequence

    The user selects a folder on the disk, all images inside the folder are measured.

    Multi Point ND File

    Each frame of a multi-point ND file is measured. An ND file is a single file containing a sequence of images (see Multi-point Acquisition). Path to the multi-point file can be specified using the Open... button.

    Current Document

    Only the current image is measured. It is useful for testing of settings before an automated measurement is run.

  4. Click on the Start Measurement button, select a detection preset, adjust the detection.

    Every measurement in a Wizard Mode starts with cleaning all previous data. In this mode it not allowed to merge old measurement with a new one. Following message appears.

    Figure 1252. 

    Click Yes to delete the content of the Data panel and continue with the measurement.

  5. The following window appears:

    Figure 1253. 

    Select a detection preset and, if needed, adjust its settings in the Detection panel.

    Click Measure Field to continue.

  6. Measure the first image.

  7. Decide whether to repeat the procedure for another image or whether to measure the rest of the images automatically.
