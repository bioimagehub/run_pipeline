Triggered Acquisition

Triggered Acquisition requires a wavelength switcher. If only the monochromator is used, it is necessary to connect another device (real device or simulator). We recommend connecting ‚ÄúNikon C-LEDFI Simulator‚Äù. This simulator enables the Triggered Acquisition functionality without occupying any NIDAQ ports or requiring any additional HASP bits.

In triggered acquisition only one power value is used for all defined channels. To be able to use multiple power setups, save each power value together with a new optical configuration ( Calibration > New Optical Configuration ).

To correctly display the wavelength values inside View > Acquisition Controls > Triggered Acquisition , click Advanced Settings..., check Use more lines for one channel and make sure PolychromeV (Monochromator) is checked in the Use these device section. Confirm the settings and simply switch to PolychromeV in the second column of the Triggered Acquisition table.

Note
