Acquisition and Analysis Modules

3D Measurement

If you have this module enabled in your hardware licence, some extra commands and features for analysis of multiple binary layers over Z become available in the Binary menu.

See also: 3D Measurement.

3D Tracking

This module is a combination of Advanced 2D Tracking and 3D Measurement.

See Tracking 3D.

6D

This module enables users to combine all 6 dimensions (X, Y, Z, time, wavelength, large image) within a single ND acquisition experiment.

Advanced 2D Tracking

This module extends the basic OBJ TRACKING module by the ability to track binary objects and by other features.

See Tracking.

Assay Launcher

This module introduces three predefined GA3 recipes to the Analysis Explorer. Additionally, if the General Analysis 3 module is installed, it adds a shortcut to the Analysis Explorer in the shortcuts section of the compact layout.

Bio Analysis

This module enables the user to perform bio-analytical tasks. The Bio Analysis module measures and analyses time-lapse and multipoint ND images (Z-stack data sets are not supported).

Several analysis modules are available - see Bio Analysis Modules.

CA FRET

With this module, you can capture/create FRET images, measure Calcium ion concentration or make automatic FRET calibrations.

See FRET.

Cell Stretching

(requires: JOBS Editor) This module enables the user to perform a cell-stretching experiment using the JOBS Editor module and a specialized piezo device (Physic Instrumente E 545).

EDF Module, Extended Depth of Focus

The EDF module allows you to combine an existing Z-stack of images into one focused image by picking the focused regions from each frame and the pieces together.

Please see Extended Depth of Focus.

General Analysis

This module enables the user to define his own image analysis procedure in a simple-to-use dialog window ( Image > General Analysis and Image > General Analysis RGB).

General Analysis 3

This module enables the modular tool for building image analysis recipes.

See General Analysis 3 .

High Dynamic Range

This module brings the functionality of capturing and creating HDR images within NIS-Elements.

See HDR.

HC Template

It is a simple-to-use acquisition and high content analysis module. Using a pre-defined template, you can acquire image data-sets, perform high content analysis and browse the resulting data.

See HCA.

Illumination Sequence

This module provides an easy graphical user interface for setting and running advanced real-time (triggered) acquisition experiments.

See Illumination Sequence

JOBS Editor

JOBS is an environment aiming at complex acquisition and data analysis with easy to use and comprehensive user interface. JOBS is a ‚Äúvisual programming‚Äù tool enabling non-programmers to design their own automated acquisition / analysis experiments. This module allows user to create, edit and run JOBs.

See JOBS and HCA.

JOBS Remote Database

With this module, you can easily share the JOBS database. Typically, it is used on multi-workstation systems where one PC is used for data acquisition and another for performing analyses.

See JOBS and HCA.

JOBS Viewer

This module enables user to browse the data acquired via JOBs.

See Running a Job, Viewing Results and Graphs.

N-STORM Analysis, N-STORM Offline Analysis

Support for N-STORM acquisition and molecule analysis.

See N-STORM, View > Analysis Controls > Molecule Analysis .

NIS.ai

An artificial intelligence (AI) module that utilizes deep learning and neural networks to automatically perform image processing tasks. The module comes with some pre-trained networks but also provides tools for training of your own neural networks.

See NIS.ai.

Object tracking, OBJ TRACKING

Motion characteristics of objects such as living cells can be measured automatically or manually using the Tracking module. Tracking is based on ROIs.

See Tracking.

Object Classifier, OBJ CLASSIFIER

Provides the ability to classify binary objects.

See Object Classifier.

Outprocess Assay

Enables the Analysis > Out Proc task in Jobs.

RT Acquisition

This module is used to perform fast hardware-driven (triggered) acquisition experiments.

See Triggered Acquisition.

Single Particle Tracking, SPT Analysis

Single Particle Tracking (SPT) is used to track single molecules and for analysing parameters of these tracks.

Please see: Single Particle Tracking.

Smart Experiment

An artificial intelligence-based module for simple management of advanced experiments performed with the Nikon Ji microscope.

Please see: Smart Experiment.

Time Measurement, TMEAS

The time measurement tool enables the user to record average pixel intensities within Regions Of Interest (ROIs) during a time interval. Please see Time Measurement.

Wound Healing - Conventional

Wound Healing analysis support.

Deconvolution Modules

2D Deconvolution

Image deconvolution module suitable for standard 2D images. Apart from the 2D deconvolution, it provides a Fast Deconvolution and Live De-Blur (applicable on live image).

Please see: Deconvolution

3D Deconvolution

Image deconvolution module suitable for image sequences containing a Z dimension.

Please see: Deconvolution

Industrial Modules

Concrete

(requires: Local Option)

This module is for automated and semi-automated measurement of air void characteristics of hardened concrete.

Filter Particle Analysis

This industrial module is aimed at analysis of filters according to the ISO 16232 standard.

See Filter Particle Analysis.

Layer Thickness

This module adds the Applications > Layer Thickness Measurement > Measure Layers command to the main menu. The module can perform different types of layer thickness measurement.

Please see Layer Thickness Measurement.

Metalo - Cast Iron Analysis

After the module installation, the Applications > Metallography > Cast Iron command appears in the main menu. Run this command to display the cast iron measurement layout.

Please see Metalo - Cast Iron Analysis.

Metalo - Grain Size Analysis

Adds the Applications > Metallography > Grain Size command to the main menu. The command displays a layout dedicated to grain size measurement.

Please see Metalo - Grain Size Analysis.

Measurement Sequencer

With this module, you can:

  • Create custom measurement sequences.

    See Measurement Sequencer.

  • Install predefined measurement sequences suitable to measure welds of different shapes.

    See Weld Measurement.

  • Install predefined measurement sequences suitable for hardness measurement (Vickers, Brinell, Knoop)

Device Modules

Confocal AX

Support for Nikon Confocal Microscope Ax.

Back Aperture Observation

(requires: Local Option) Support for an additional camera, it displays its live image and FFT image in a separate window.

Digital Mirror Device

(requires: Local Option) DMD support. E.g.: Nikon Ti-Lapp, Andor Mosaic 3, Mightex Polychrome 400.

Dual Camera Support

Support for dual camera drivers (two mono cameras behaving as a single dual-channel camera). Specific camera models are supported.

FLIM

(requires: Local Option) FLIM support

Monochromator

(requires: Local Option) Thirdparty monochromator support. E.g.: Till Photonics Polychrome V.

NSIM

Nikon N-SIM support.

NSIM Offline

Offline Nikon N-SIM support.

Optical Tweezers

(requires: Local Option) Thorlabs optical tweezers support.

Optogenetics Acquisition

(requires: Local Option) A plugin used for acquiring two image sources - confocal A1 and Imaging Source - to one image.

Revolution DSD

(requires: Local Option) Andor Revolution DSD confocal support.

Slide Loader

(requires: Local Option) This module enables NIS-Elements connect and control a slide loader.

SFC

(requires: Local Option) Swept field confocal support.

Shutter Filter Wheel

Third party filter-wheels support.

SPD_X1

Yokogawa Spinning Disc (CSU-X1) support.

SPD_W1

Yokogawa Spinning Disc (CSU-W1) support.

Splitter DV

Dual View and Quad View support.

See also Using a ‚ÄúMultiview‚Äù Device.

Stage

This module enables the user to connect and control motorized XY stages. All non-Nikon stages require this module.

Stage Incubator

This module enables the user to connect and control gas and temperature-controlling chambers.

Structured Light

(requires: Local Option) Support of structured-light-based devices (OptiGrid, ViCo).

TTL/Analog IO

This module enables NIS-Elements to send and receive TTL and analog signals via a DAQ board.

See also Triggered Acquisition.

Temperature Control

(requires: Local Option) Support for heating stages.

Tripple/Quad camera support

(requires: Local Option) Update from Dual Camera Support. This module supports up to four mono cameras behaving as a single multi-channel camera.

Wavelength Switcher

This module enables NIS-Elements connect and control wavelength switchers.

See also Illumination Devices.

Well Plate Loader

This module enables NIS-Elements connect and control a well plate loader.

X-CITE

Third party illumination device support.

XY Galvo device

(requires: Local Option) Third party galvo devices support (E.g.: Bruker miniscanner).

Z Drive

This module enables connect and control motorized Z drives.

See also Connecting a Device to NIS-Elements.

Other

DBASE (Database)

If the database module was installed, the system can create and control image databases. A database can help you to efficiently organize your image archives and manage additional information related to the images.

Enhanced Resolution

Extra functions such as Automatic Deconvolution are added with this module. See NIS-Elements ER Package.

Local Option

When Local Option is installed (see Step 2), NIS-Elements will provide some advanced features which did not pass the quality assurance procedure yet. We recommend to wait until they are released officially.

MS Excel

NIS-Elements supports export of results straight to an MS Excel sheet (see Exporting Results). MS Excel is not part of the NIS-Elements and must be purchased and installed separately.
