File > New

Creates a new report or report template.

File > Open

Opens a report (*.rpt) or a report template (*.rtt).

File > Save

Saves the report to disk.

File > Save As

Saves the already saved report (or template) under a different name.

File > Create Report

If you are in the template mode, this command processes the template and creates a report using the data available (e.g. the currently opened image).

File > Change to Template

Use this command to switch from the report mode to the template mode. Template mode enables the user to define dynamic objects which will be later filled with data upon report creation. See Report Templates.

File > Open Source Template

Each report created from a template remembers the path to the template. Use this command to open the report template which was used to generate the currently displayed report.

File > Page Setup

Set dimensions of the report page so that the resulting page will be compatible with the intended output (such as a printer). Measurement units can be changed via the Edit > Options command.

Paper size

Select one of the predefined paper sizes or specify a custom Width and Height.

Width, Height

Specify width and height of the page.

Borders

Define position of the dashed guides displayed on each page.

Note

You will be still able to place objects anywhere on the page, but the guides are used for automatic alignment of objects with the option Relative to Page turned on.

Orientation

Choose orientation of the page, the Width and Height values will switched when you change the orientation.

File > Print

Opens a system dialog window where you can send the current report to an installed printer.

File > Send

Opens a default e-mail client window and allows to send the current report by e-mail.

File > Export to PDF

Saves the current report as a PDF file. Select a folder and specify the PDF name, then click Save.

File > Export to RTF

Saves the current report as a .rtf document file. Select a folder and specify the file name, then click Save.

File > Close

Closes the Report generator.
