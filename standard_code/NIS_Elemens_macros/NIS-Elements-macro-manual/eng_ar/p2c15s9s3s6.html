Adding Metadata to Loops

Each loop type has a different set of metadata which must be assigned using Define....

Figure 1046. Defining loop metadata

The definition dialog is partially filled by default. The non-parsed columns could be edited. Clicking on Refill inserts the default values.

Figure 1047. Position Loop definition

In the Define Position Loop dialog, the position of each point should be filled. Positions are supposed to be set in the ‚Äúlarge image coordinate system‚Äù. For example when there is no overlap, the step in x-axis direction between the neighboring frames can be calculated as frame width * calibration (see the image below). Positions can also be imported from MS Excel clipboard using the Paste from Clipboard button in the toolbar of the definition dialog.

Figure 1048. Coordinate system for Position Loop

Figure 1049. Channel Loop definition

In the definition dialog for a well loop, a well-plate must be selected in order to be able to correctly parse the well position.

Figure 1050. Well plate definition

If all the needed information is filled, the status bar at the bottom of the main dialog shows the ‚ÄúLoops are correctly defined.‚Äù message and OK is enabled. Otherwise warning messages are shown.
