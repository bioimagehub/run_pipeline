Stg_Filter_GetPositionName

Description 

 Stg_Filter_GetPositionName(
   char *FilterChangerName,
   int  FilterPosition,
   char *LpFilterName,
   int  FilterNameMaxLength
);

Returns a filter position name.

Parameters

char *FilterChangerName

Name of the filter changer - the same name which is displayed in the Device Manager.

int FilterPosition

Index number of the filter changer position.

char *LpFilterName

Name of the filter changer position.

int FilterNameMaxLength

Specify the maximum number of characters of the name.

Return Values

DR_OK (1)

Filter name was stored to name buffer

DR_BADPARAMETER (-2)

Some parameter has invalid value

DR_NOTAVAILABLE (-4)

Device Filter Changer is not present

DR_CANCELED (0)

Filter name is longer than length of name buffer

DR_UNKNOWNERROR (-1)

Retrieving the filter name failed

See Also 
Stg_IsFilterPresent, Stg_GetFilterPositions
