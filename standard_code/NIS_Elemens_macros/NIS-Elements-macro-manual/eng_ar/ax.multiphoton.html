Creating a Multiphoton Experiment Preset - DUX Spectral Detection

(requires: AX MP)

  1. Select the AX MP DUX lightpath in the Acquisition panel.

  2. Open the Experiment Setup window by the Exp. Settings button.

  3. Two types of presets can be created here.

    DUX Multi-Channel

    Analogy to the Advanced presets if IR lasers and DUX detector are used.

    DUX Spectral

    (requires: DUX-VB)

    Analogy to the Spectral presets if IR lasers and DUX detector are used.

  4. Click the Add... button and proceed as described in the sections:

    • Creating a Multichannel Experiment Preset

    • Creating a Spectral Experiment Preset

  5. Save the Preset

    When everything is set, click on the Save As button, type a name of the new preset to the window and confirm it by OK.

    Click the Close button to close the window.

    Note

    If the assignment of filters was changed, a confirmation dialog appears (Changing of Filters in the NDD detector).

Creating a Multiphoton Experiment Preset - NDD Detection

(requires: AX MP)

  1. Select the AX MP NDD lightpath in the Acquisition panel.

  2. Open the Experiment Setup window by the Exp. Settings button.

  3. Specify fluorescent dyes

    Similarly as in a multi-channel preset, add the dyes which represent the wavelengths which will be acquired.

    • Click the Add... button next to the list of NDD imaging presets.

    • Search for the dyes and add them to the preset.

    • Click Finish to close the window.

  4. The software assesses your selection of dyes and creates the preset with reasonable options selected.

    Figure 183. DAPI+FITC NDD experiment defaults

    You can adjust them arbitrarily:

    • Lasers can be re-assigned to different channels.

    • Click the filter-cube icon to select a different cube.

      Figure 184. Edit NDD filter selection

    • Channels can be added by the + button.

    • You can use the Filter cubes to load default parameters for all the selected filter cubes.

  5. Save the Preset

    When everything is set, click on the Save As button, type a name of the new preset to the window and confirm it by OK.

    Click the Close button to close the window.

    Note

    If the assignment of filters was changed, a confirmation dialog appears (Changing of Filters in the NDD detector).

Changing of Filters in the NDD detector

(requires: AX MP)

Every time, the assignment of filters in the NDD detector is changed (e.g. if you select a different experiment setup in the Experiment), the following window appears which requires the user to change the filters safely.

Figure 186. NDD filter change - confirmation dialog

To change filters in the NDD detector safely:

  1. Turn the Protect NDD switch ON. It sets gains of the PMTs to 0 in order to prevent their damage when opening the NDD detector.

    Figure 187. 

  2. Change the filters in the NDD detector manually and close it.

  3. Turn the Protect NDD switch OFF.

  4. Confirm the new configuration by clicking the Confirm button.
