Window > Next

This command switches to next opened image.

Window > Previous

This command switches to previous opened image.

Window > Close All

This command closes all opened documents in NIS-Elements.

Window > Close All but Current

This command closes all opened documents, but keeps the current one opened.

Window > Close

Equals the File > Close command.

Window > Cascade

This command sorts all opened documents to a cascade.

Window > Tile Horizontally

Covers the available area by tiles of all opened images. The images are ordered to columns which, in some cases, cause horizontal orientation of the image windows.

Window > Tile Vertically

Covers the available area by tiles of all opened images. The images are ordered to rows which, in some cases, cause vertical orientation of the image windows.

Window > [Opened file]

This command brings the selected image into focus (makes it active).
