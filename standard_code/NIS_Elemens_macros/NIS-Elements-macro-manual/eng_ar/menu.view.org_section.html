View > Image Layout

Closes the Organizer and returns to the program main screen.

View > Hide Folders

Hides the folder tree within the active panel.

View > Hide Form

Hides the database form within the active panel.

View > Show Folders

Displays the folder tree within the active panel.

View > Show Form

Displays the database form within the active panel.

View > Reorder Tables and Databases

Displays a dialog window where you can change order of databases and tables (within each database). This order is used in the main toolbar selection menus.

Databases

List of connections to databases. Select one item and use the arrow buttons to move it up/down the list.

Tables

Once a connection to a database is selected, tables of this database are listed. Re-order the list in a similar manner.

View > Maximize Left Pane

Maximizes the left panel to the whole window (hides the right panel).

View > Maximize Right Pane

Maximizes the right panel to the whole window (hides the left panel).

View > Set Panes Same Size

Adjusts widths of the left and right panel to 50% each.

View > Icons

Sets size of the displayed image thumbnails for the active panel.

View > Small

Sets size of the displayed image thumbnails for the active panel.

View > Medium

Sets size of the displayed image thumbnails for the active panel.

View > Large

Sets size of the displayed image thumbnails for the active panel.

View > Very Large

Sets size of the displayed image thumbnails for the active panel.

View > Details with Preview

Displays image thumbnails of the active pane in a table along with image/database information.

View > Next Pane

Switches activity between panels.

View > Refresh Folders

Use this command to reload the list of images displayed within the active pane. For example, run this command if images were moved, renamed or otherwise modified by an external program.
