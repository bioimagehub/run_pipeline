HCA Analysis

(requires: HC Template)

NIS-Elements provides the following analysis modules:

  • Cell Count Analysis

  • Image Intensity Analysis

  • Cell Motility

  • Sperm Motility

  • Wound Healing

  • General Analysis

  • Image > General Analysis RGB

  • Live/Dead Analysis

You can include one or more of these analysis modules to be a part of your HCA job or you can just use them to process the resulting images after running the acquisition part of the job.

Analysis performed on job results

  1. Go to the Job Results window e.g. by double-clicking the job run record.

  2. In the result view (Running a Job, Viewing Results and Graphs) , click the Run Analysis button.

  3. The Run analyses dialog window opens.

    Figure 917. 

    Click Add New and choose the type of your analysis from the pull down menu. You can change its name by double-clicking in the Name column. Select which portion of the data will be used for the analysis run (Run on column). A selection of frames can be made prior opening this dialog. Click Define... and continue setting up your analysis (described in separate analysis chapters). Once the analysis is set up correctly a green icon is shown. To import analysis definition from a recipe use Load. Duplicate duplicates the selected analysis whereas Delete removes it from the list. Define as many analyses you like and once you are done click Run. Your analyses are then processed and the resulting variables are added to the Show Quantities panel.

    Tip

    Hide your preview by clicking the Show preview button to see more of your analysis results.
