Sequential Measurement

  1. Place the images that you want to analyze sequentially to one folder.

  2. Select the Measure > Measure Sequence command.

  3. A window appears. Set the right Images of Type that will be displayed in the images list.

  4. Select the images to be analyzed with a mouse: one by one holding the Ctrl key down, or select a range holding the Shift key down.

  5. Run the batch process by clicking the Measure button.

  6. Confirm each single measurement the same way as if working with a single image.
