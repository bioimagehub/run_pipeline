Measurement of Tracking Features

A data set of object features is gathered during the tracking procedure. Common static and dynamic features of objects can be measured (e.g.: object area, object position, acceleration, velocity, ...).

Description of each supported tracking feature can be found here: Measurement Features.

Which features are measured is determined in the Object Tracking Settings dialog window. To display it, click on the Data tab in the Tracking panel, right-click on one feature in the table and choose Settings....

Figure 1408. 

The check boxes indicate which features are going to be measured and shown in the Data table.

Figure 1409. 

Define Reference Point

There are three movement-descriptive features which can be measured in relation to a reference point (if it has been defined). These are the Reference Length, Reference Velocity, Reference Acceleration. If an object was moving in circles around the reference point, the features will be constant / zero.

  • Press the Reference Point button to display reference point in the image.

  • Position of the reference point can be changed by drag and drop.

Export 
The measured data or the graph can be exported to an external file using the Export button.

Please refer to the Exporting Results chapter for further details.
