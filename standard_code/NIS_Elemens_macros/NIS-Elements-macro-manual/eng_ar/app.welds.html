Weld Measurement

Basic Workflow

(requires: Measurement Sequencer)

The Measurement Sequencer module can be used for analysing images of the following types of welds:

  • Fillet Weld

  • Lap Weld

  • Spot Weld

Basic Workflow

  1. Display Measurement Sequencer

    Run the Applications > Weld Measurement > Measurement Explorer command to display the panel for running measurement sequences.

  2. Select a Measurement Sequence

    There will be different measurement sequences in the list for different weld shapes (Lap, Fillet, etc.). The default templates are displayed in red color. Double-click the template name - e.g. Fillet Weld. The Measurement Sequencer - Run panel appears.

  3. Select parameters

    Select check boxes next to the parameter names for those parameters to be measured.

  4. Start measurement

    Make sure the Show Drawing button is pressed so the scheme of the current weld shape containing dimensions is displayed. Open the image for measurement. Click the Measure button.

  5. Draw the required lines and dimensions to the image

    Dimension lines appear in the drawing to give you a hint of how to measure the current parameter correctly. A textual hint may be displayed below the drawing. Click inside the image to measure the first parameter. When finished, the sequence will proceed to the second parameter, etc.

  6. Save results

    After the measurement is finished, the Export to Report gets enabled. Click it to save the result.

  7. Proceed to another image

    To measure another image using the same settings, open it and click the Restart button.
