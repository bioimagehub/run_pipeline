System Info

Click the Configure button on the AX Pad panel to display the settings window. There are several tabs:

Acquisition

Figure 206. 

Galvano Scan Direction, Resonant Scan Direction

Sets the preferred scan direction. Unidirectional scan or bidirectional scan is available for each scanner. The bidirectional mode is faster but needs to be aligned properly in the Alignment tab. If this mode is misaligned, a message and info button will appear, allowing you to click the button and align it Automatically or Manually.

Caution

In Galvano mode, bidirectional scanning is reserved for resolutions 1024 x 1024 and smaller. You can select the bidirectional mode here, but when higher resolutions will be set, the scanner will switch to a unidirectional mode automatically.

Average/Integrate Mode

Select the noise-reduction method and the range of frames which will be available in the AX Pad panel.

Line Scan 1DT
Automatic number of lines

The automatic number of lines is 512 for all line lengths. De-select this option to set the size manually.

Capture 1DT number of lines

This value is used with the Acquire > Capture command.

ND Acquisition 1DT number of lines

This value is used in ND Acquisition experiments.

Find Mode
Automatic

In this mode, the Find mode acquisition parameters are calculated to achieve the fastest possible frame rate. A pre-trained AI network is used to enhance the low-resolution live image. A well aligned bi-directional mode is recommended.

Manual

Optionally, you can set the find-mode parameters manually. In this mode, no image enhancement is applied.

Bidirectional Scan

Use bi-directional scanning mode.

Line Skipping

Frame rate can be increased by skipping lines. Line skipping 2x scans every second line, line skipping 4x scans every fourth line.

LUD-S Shutter Mode

If this option is displayed, your laser unit is equipped with a controllable mechanical shutter.

Electronic

A mirror is used to redirect the lasers out of the optical path when the device is idle. Minor intensities could reach the sample in this mode even though.

Mechanical

Choose this option to prevent any light to reach the sample during idle phases of work.

General

Figure 207. 

General Settings
Keep Pixel Brightness (Galvano)

This option allows you to acquire images with their brightness unchanged even if the Pixel Dwell Time is changed. It works only with the galvano scanner type.

Apply Gamma Scaling

This function uses a gamma curve to lighten dark areas of the image. This option is only applied if the High Dynamic Range option is selected.

High Dynamic Range

The original (wider) dynamic range of the detectors is re-calculated to the 12-bit range of the resulting images. With this option ON, the whole range of the detector is used, otherwise, the range is slightly cropped.

Hide Offsets and Always Apply Value

Hides sliders in the AX Pad panel and applies the offset values specified here. Different offsets can be specified for galvano/resonant modes and the TD channel.

Channel Selection via Left Mouse Button

If selected, channels can be turned ON/OFF arbitrarily in the Experiment panel.

Gain Behavior during Stimulation

Detector gains can be controlled automatically during stimulation experiments in order to handle the case when a PMT overload occurs.

Zero Gain

Sets the gain on all detectors to 0 during stimulation. The original gain is restored after stimulation. PMT overload cannot occur.

Keep Gain

The automatic gain control is turned OFF - the gain remains unchanged. If PMT overload occurs, a red alert appears in the AX Pad panel. The user must adjust the gain manually to restart the detector.

Auto Recovery

The gain remains unchanged. A PMT overload can occur, but the detector is recovered automatically after the stimulation.

Shading Correction
Disable and Don't Show

Hides the Shading Correction button on the AX Scan Area panel. Does not apply any correction.

Show on AX Scan Area

Displays the Shading Correction button on the AX Scan Area panel.

See Applying Shading Correction.

Framerate Text Display Mode

Select the format how the FPS and Frame Time values are displayed on the AX Pad panel.

Alignment

Figure 208. 

When bidirectional scan mode is selected in the Acquisition tab, a shift between even and odd lines may occur in the image. You can correct the shift here.

Only the active scanner can be aligned, if the buttons are disabled, close the window and activate the other scanner in the AX Pad panel.

Activate Manual Alignment

Starts scanning and displays the live image window. You can check influence of the alignment.

Auto Align, Auto Align X, Auto Align Y

Runs a dedicated algorithm which calculates the pixel gap automatically based on a series of captured images. Make sure the sample for automatic alignment has distinct texture and minimum noise.

Note

The galvano scanner enables rotation of the scan area. Therefore alignment of the Y direction is performed with the scan area rotated by 90 degrees.

Pixel Gap

You can adjust the pixel gap manually. Either drag the slider by mouse or type a value to the edit field on the right.

Sampling

Figure 209. 

Keep Pixel Size (XY)

Determines behavior of the Nyquist / Optimal button which adjusts scan area parameters automatically in order to achieve optimal pixel size. When the button on the AX Scan Area panel is pressed:

Nyquist

The scan size remains unchanged and the zoom size value is adjusted automatically.

Optimal

The zoom size remains unchanged and the scan size (image resolution) is adjusted automatically. The resolution which provides the closest value of the pixel size to the Nyquist resolution will be selected.

Use Cropping

This function allows users to focus on a sub-area of the scan while maintaining the original pixel size. If the selected resolution is larger than the current resolution and Use Cropping is checked, the image is cropped to preserve the resolution. This ensures that the image pixel size remains consistent, and a new pixel size is calculated based on the cropped area.

Suggested Step (Z)

Select how the suggested Z-step value is computed in the ND Acquisition Z series settings window.

It is computed as a ratio of the optical section thickness (optical section thickness = FWHM of the Z airy disc).

Pinhole

Figure 210. 

AU Setup
Laser

Select a wavelength from which the AU is calculated.

Shortest Î», Medium Î», Longest Î»

One of the lasers used in the experiment will be used.

Home Position

Sets a default position for the pinhole. This pinhole position is set by the button in the AX Pad panel.

Custom Position

Sets a user-defined pinhole position and displays an additional shortcut-button in the AX Pad panel. If the value is 0, no button will be displayed.

Pinhole Alignment

The pinhole position can be aligned by manually shifting the condensing lens.

Coarse, Fine

Use these edit fields to define step for the arrow buttons.

Lens Position

Change the X and Y lens position by either clicking on the arrow buttons, typing inside the X and Y edit fields or clicking on the spin buttons.

Live

Display the live image by this button to see how the alignment affects the image.

Note

Different alignment settings are remembered for each position of the 1st dichroic mirror and different alignment is also used if the second laser port of the AX head is used (FLIM/FCS applications).

NIDAQ Interface

The piezo switcher box allows you to trigger a piezo Z stage by the AX controller or a camera without re-connecting of cables.

Piezo Z Analog Line

Switcher Box TTL

Triggering

Figure 212. 

Each column represents values used with different Autosignal.ai setting.

Image Quality [0-24]

Select the preferred image quality for each setting.

Note

The higher the number is, the higher is the risk of bleaching the sample. Make sure the values are low-enough not to ruin your sample but high-enough to get as good an image as possible.

Maximum laser power which can be set by the AutoSignal.ai button.

IR Laser Limits - Tunable

(requires: MP)

Maximum power or the IR laser which can be set by the AutoSignal.ai button for the default wavelength. For other wavelengths, the power is recalculated according to the laser calibration curve. See Defaults-MP.

AutoFocus

Capturing Mode

The default option, Find Acquisition (faster), enables AX auto focus to use the Find mode for acquisition. In this mode, AX checks the FPS, and if it is below 7, it reconfigures the system for faster performance. To revert to the original behavior, select Regular Acquisition, which applies the same settings used for the standard Live/Capture modes.

(requires: MP)

Figure 213. 

IR Laser Control
Show on Pad

Select this option to display the IR laser control on the AX Pad panel.

Emission, Shutter, Humidity

See the description here: IR Laser Control.

Servo

Option of the Insight X3 IR laser. If periodic noise has been observed during sensitive measurements, the servo may be turned off for future measurements. It is recommended to turn the servo back on as soon as a measurement is completed.

Note

The servo will be automatically turned on after 60 [min] in order to maintain laser performance.

IR Laser Alignment

Optical XY alignment of the IR laser is different for each laser wavelength. There is an automatic alignment mechanism which should be run every time the wavelength is changed.

Automatic

If selected, automatic XY shift alignment will be performed after the IR laser wavelength is adjusted.

Auto Alignment

Click this button to run automatic alignment at once.

Or, the shift can be adjusted manually by the arrow buttons.

IR Parfocality Adjustment

Parfocality of the IR laser can be adjusted by the user.

Current

Run the live image. Click on the arrows to adjust parfocality offset for the current objective.

  • If you close the dialog, the value remains set

  • If you change the objective or laser wavelength, this value will be reset to default (see below).

Default

Once you achieve a satisfactory setting of the Current text field, click the Set button to remember this value as default.

  • If you change the objective or laser wavelength, this value will be copied to the Current text field where it can be further adjusted.

The Remove button sets this value to 0.

Note

If the configuration contains two IR lasers (Dual IR) both with adjustable parfocality, a separate adjustment section appears for each (Main and Sub).

System Info

Figure 218. 

System Version Info, Firmware

Various system information

Laser Power Monitor

Click this button to verify actual power of the installed lasers. A laser-power monitor window appears.

  1. Select laser wavelength.

  2. Click the Measure button. A new row will be added to the table.

  3. The result is a unitless value displayed in the last column. Perform this measurement whenever you suspect there is something wrong with the lasers. If the value differs a lot from the previous measurement it is a sign you should contact your local dealer of the system to request support.

Resonant Scanner

(requires: AX-SHR)

The resonant scanner produces noise which you may want to suppress. When the Resonant sound reduction function is used, the scanner resonates only during scanning (Live, capture image, ND Acquisition). Select when or whether to turn the resonant scanner OFF.

Mute immediately on idle, Mute N seconds after idle

The disadvantage of these options is that a ~1 second delay needed to stabilize the scanner resonance will occur when starting live image or acquisition.

Do not mute

The noise is not suppressed.

Turn Resonant OFF when Muted (slower acquisition start)

Muting only reduces frequency of the scanner. This option turns off the scanner completely. However, the delay before the live image will be longer by seconds.

Dichroic Mirror, Detector Filters

These sections display the current filter setup of the 1st dichroic mirror and the detector filter wheels. You can re-assign the filters if you change them inside the device. The fill colors represent reflected wavelengths and the smaller rectangles below them represent transmitted wavelengths.
