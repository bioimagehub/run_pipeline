Automatic Capture

This mode measures Grain Size on a requested number of fields inside a defined area. When selected, the following window appears:

Note

The scanning stage must be initialized beforehand.

Figure 1254. 

  1. If you started the Grain Size Wizard and selected the option Automatic Capture as the image data source, continue as follows.

  2. Live camera image appears overlaid by a red cross. Use the arrow buttons or a joystick to move the sample so that the red marks the top left corner of the scanned area.

    Figure 1255. Stage Area Preview

  3. Specify the Sample size and click the Check Bounds button. The XY stage will move the red cross to the bottom-right corner of the sample. Check whether the red cross marks is placed as expected, otherwise modify the specified sample size accordingly.

  4. Specify the number of fields to be scanned (Field Count). XY positions are calculated automatically by the software.

    See descriptions of all options in Grain Size Wizard - Start.

  5. Click the Start Measurement button. The first field will be captured and a window for defining the detection parameters appears.

  6. Select a Detection Preset and click one of the buttons, or , to check the detection. If needed - adjust options in the Detection panel - especially the threshold.

    Figure 1256. 

    If the preset is modified and not saved under a new name, the changes in detection are stored automatically to the actual preset.

    See also Grain Size Wizard - Select Detection Preset.

  7. Click the Measure Field button. A window appears where you can select a Measurement Preset.

    Figure 1257. 

    See Grain Size Wizard - Select Measurement Preset

  8. If needed, adjust settings in the Measurement panel, the changes will be saved to the preset in the next step.

    If needed, click the Back to Detection button to return to the detection adjustment. Then continue with the Measure Field button again.

  9. Click the Done & Next Field button to finish measuring the first image.

  10. It is recommended to repeat this procedure (capture image, check the detection, check the measurement) at least twice. If you are confident that the settings are fine, click the Done & Measure All Fields button. The rest of fields will be measured using the most recently saved presets.

  11. View the resulting data and export them in the required format. See Data output and Reports.
