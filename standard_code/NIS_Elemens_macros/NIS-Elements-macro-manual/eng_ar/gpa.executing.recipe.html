Executing a recipe

To run an existing GPA recipe, select a GPA definition from the GPA Explorer list (GPA Explorer) and click Run or double-click on it. GPA run tabs will appear in the middle section of the application. Click and adjust all the tabs settings step-by-step as described below.

Figure 493. GPA Run.

Light

  1. Enter the Sample ID in the field below the stage preview and adjust the light source (fit into the limits if Require image intensity level was set for the current recipe).

  2. Click Next.

Sample

  1. Move the green cross in the live image to precisely position the sample corner/edge and use the buttons below the stage preview to correct this position.

  2. Click Next.

Focus

  1. If Focus Plane or Focus Plane and Auto Focus were set in the Focusing Options of the current recipe, then click on a point inside your sample area, precisely focus on the sample and click Set and Next.

  2. Repeat this procedure for the second point not too close to first one. Click Set and Next.

  3. Repeat this procedure for the third point not too close to the previous two points and click Set and Finish.

  4. Move to a random point and click Move to Focus Plane to verify that it is really in focus or use the Move to Focus Plane and Auto Focus to do the focus automatically.

  5. Click Next.

Capture

  1. Automatically scans the image. No user interaction is required.

Detect

  1. Adjust the segmentation levels so that your particles are highlighted in the live image with the chosen segmentation color.

    Note

    If the segmentation is disabled, click Check Detection.

  2. Click Next.

Measure

  1. Measures / captures the sample and finishes the GPA run with a new record in the GPA Explorer.

  2. Click Next when the measurement is done.

Inspect

  1. If an Inspection step was added in the GPA Definition then the GPA Inspection interface opens.

    Figure 494. GPA Inspection.

    Here you can check and correct the detected objects using the Draw and Erase tools in the left toolbar to further improve the ongoing segmentation.

  2. Choose a binary layer which will be modified by drawing (select the rectangle of the binary layer below the image).

  3. Select a tool, adjust the line thickness (Size) and then draw directly into the binary layer in the image to correct the first object.

  4. Once your object is corrected, click OK & Next.

  5. Correct another object and click OK & Next. Repeat this procedure until all your objects are correctly segmented.

    Note

    To skip correcting objects at any time, click Finish. To correct a class of objects or all objects again, click Re-inspect.

    Tip

    Click Show Contour to easily see the edge of the binary layer of the object. Use Ctrl+up/down arrow keys to increase/decrease the binary object transparency.

  6. After the last object is inspected the measurement is repeated utilizing the segmentation improved by the manual inspection and the results are shown in the bottom part of the application.

Tip

To repeat the measurement procedure on the same GPA definition, simply click on New.
