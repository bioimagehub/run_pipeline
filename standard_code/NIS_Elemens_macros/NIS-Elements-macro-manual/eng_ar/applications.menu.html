Applications > 6D > Define/Run Sequential Stimulation

(requires: 6D)

This command displays a window where it is possible to setup a sequential stimulation/bleaching experiment.

Figure 1872. 

Columns

To specify the experiment, you have to define one or more phases (rows) and adjust values in the following columns:

#

Phase number. To add a new phase, click inside the check box.

Group

You can group several phases of the experiment and set number of repetitions. Select two or more adjoining phases (holding Shift) and press the Group button. Then, double-click the first row and specify the number of repetitions.

Acq/Stim

In this column, specify the phase type: Acquisition, Stimulation/Bleaching (these two are equal), Waiting.

Interval

Specify the interval between two actions of the phase.

Duration

Specify the overall length of the phase.

Loops

Specify the total number of actions of the phase.

Note

The Interval, Duration and Loops columns are dependent. If you specify two of them, the third will be calculated automatically.

Commands

Perform Time Measurement

Check this item to let the application automatically perform Time Measurement.

Close Active Shutter when idle

Check this item to close current active shutter in time between two captures.

See About ND Acquisition for description of the dialog options common to ND acquisitions.

Applications > 6D > Define/Run Simultaneous Stimulation

(requires: 6D)

This command displays a window where it is possible to setup a simultaneous stimulation/bleaching experiment. Define the two simultaneous actions within the control window and press Run now.

Figure 1873. 

Acquisition

The interval parameter is always set to No delay. Set either Duration or Loops in order to specify the length of the captured time-lapse image.

Stimulation/Bleaching

The Wait value specifies time from the beginning of acquisition to the start of stimulation/bleaching. You can also select the Manual value to operate the stage manually during stimulation experiment. The manual option also defines the Position of stimulation ROI as static or manually repositionable. The Duration can be also set in manual mode.

Note

The Free Point Stimulation is supported only for a Stim. Point ROI. If any other stimulation ROI is used, then a warning message box is displayed when the Run now button is clicked.

See About ND Acquisition for description of the dialog options common to ND acquisitions.

See also Combined ND Acquisition.

Applications > 6D > Define/Run Parallel Stimulation

(requires: 6D)

This command opens the ND Stimulation Parallel panel allowing parallel camera recording during stimulation with a stimulation device such as a DMD (Mightex, Mosaic), Galvo XY, or FRAP. The user can choose X number of phases on Y stimulation devices, either periodically repeating or continuous. It is a software triggered process which simply scans the given ROI (or in case of DMD it shines into it) either using or not using a time interval. Collisions in the stimulation phases (e.g. one device used in multiple phases in overlapping cases) should be avoided.

Name

Name of the experiment.

Save to file

If checked, another area is revealed enabling the user to type in or Browse the path where the files will be saved. Filename specifies the name of the file and its suffix can be selected from the drop-down menu.

Record Data...

Opens the Recorded Data dialog window enabling to choose which data are to be recorded and to optionally Enable Periodic Recording with a defined frequency.

Custom metadata

If checked, another area is revealed enabling the user to edit custom metadata and use them in the experiment. Clicking on Edit... opens the Edit Custom Metadata dialog window (see Custom Metadata for details). The first drop-down menu selects the metadata set. Once it is selected, all the metadata contained in the set are loaded and can be used.

Optical configuration

Selects the optical configuration used for acquisition.

Total Duration

The total duration of the experiment can be specified either by a time interval or by a number of frames.

Acquisition Period

Sets the time interval for the acquisition period. If no interval is needed, use No Delay.

Phase

Stimulation phase area. To add another stimulation phase, click Add Phase. To delete a phase, click on X in the top right corner of the phase.

Stimulation Configuration

Clicking on ... opens a dialog for selecting a previously created stimulation configuration.

Continuous, Periodic

Sets whether the stimulation is continuous or periodic.

Start

Sets the stimulation phase start point.

Duration

Sets the stimulation phase duration.

Stimulation Period

If the stimulation phase is switched to Periodic, the stimulation period (pulsing) can be set here.

Stim. Duration

If the stimulation phase is switched to Periodic, the stimulation duration is set here.

Add Phase

Adds another stimulation phase.

Perform Time Measurement

If checked, time measurement is performed and the Time Measurement dialog window opens.

Load

Loads the experiment definition from a previously saved file.

Save

Saves the experiment definition to a file.

Run now

Runs the parallel stimulation experiment.

Applications > ImageJ Bridge > Process Current Image

Opens the Process Current Image dialog window used for image processing through ImageJ. After clicking the OK button, the following steps are performed:

  1. Saves the currently active image to the defined Input location in tif format

  2. Starts the defined ImageJ Executable (*.exe) as a new process and passes the specified Parameters and the path to the Macro to it.

  3. Waits for the process to end.

  4. Saves the Output image and opens it in NIS-Elements.

Figure 1876.

Input

Output

Applications > EDF > Create Focused Image

(requires: EDF Module)

When the method is selected and the sequence is aligned, the only thing to do is to run the Applications > EDF > Create Focused Image command. The focused image will be created and appended to the ND2 file (when the ND2 file is saved to disk the focused image is included).

Z-Map Setting

Z-Map method

Sets the method for the Z reconstruction. This reconstruction takes all Z slices and their focused image to create a 3D model of the whole scene called Z-Map. The Balanced method brings more consistent results with less noise and artifacts when compared to the former faster method (Original). If the Z-map calculation is not necessary for your task, select None to speed up the EDF image acquisition.

Zero Based

Places the lowest Z slice in the zero level, e.g. for better interpretation.

Real Coordinates

The original absolute Z coordinates remain intact.

Do not show this dialog again in this session

Check this option if you do not want to see this dialog in the current session again.

Applications > EDF > Edit Background

(requires: EDF Module)

This function enables the user to handle the background of a Z-series surface image viewed in the Show EDF 3D Surface View ( Applications > EDF > Show Surface View ). After the focused image is created, you can invoke this command. The following dialog appears:

Figure 1879. 

First, you have to detect the background. Click Auto detect to perform the autodetection. If the results are not fully satisfying, or in any other case, choose Edit manually... to open the ROI editor. Detect the background using the tools available and then press the Tab key or click Exit Editor to close the editor.

See Binary Editor.

Check Overlay and set the color of the overlay for better visualization of the edited layer.

Finally choose how to process the background in the Application section. Remove option deletes the detected background area. Flatten will set all the background Z-coordinates to zero so that the background will form the zero level which can be moved in the Z dimension Up or Down. If you do not want any of the mentioned options, select Keep original background.

Hardware Setup

The FRET control window consists of several sections. In this section, optical configurations which will be used during the acquisition should be assigned to FRET channels.

Figure 1888. 

Camera Setting

In this pull-down menu, select an optical configuration that contains Camera settings which shall be used during the FRET image acquisition. If None is selected, the current camera settings will be used.

Channels

The FRET image can contain 2-4 channels. Here you can assign an optical configuration to each of them. It is recommended that the optical configuration contains only the filter changer settings.

Dual View

If the Dual View device is available, it can be used for channel splitting. Then, only one optical configuration for each excitation wavelength will be used.

Note

The Dual View device splits the emitted light to two beams filtering different wavelengths, and each of them is captured by only a half of the camera chip.

Load, Save

The channels settings may be saved to the system by the Save button. Once saved under some user defined name, this name appears in the Load pull-down menu ready to be loaded. The settings can also be loaded from an external FRET image.
