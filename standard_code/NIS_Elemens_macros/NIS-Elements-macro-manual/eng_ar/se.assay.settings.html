Assay Settings

Opens the Assay Definition window listing all available assays. Administrator can change the enabled parameters in the right-most column for each assay. Assays are not editable by default. However a copy of an assay which can be adjusted can be created from the context menu over the assay name (Make Copy).

Note

Assay has to be enabled by the administrator (saved in this dialog) before it can be used in the Run Assay window.
