Edit > Undo

Equals the Edit > Undo command in the main menu.

Edit > Redo

Equals the Edit > Redo command in the main menu.

Edit > Cut

Cuts the selected object (text, image, table...) from the report and moves it to clipboard.

Edit > Copy

Equals the Edit > Copy command in the main menu.

Edit > Paste

Inserts the contents of the clipboard into a destination location in the report.

Edit > Select All

Selects all the objects in the report.

Edit > Deselect

Deselects all the selected objects in the report.

Edit > Options

Opens the Options dialog window containing following options:

Page unit

Defines the unit used for the object properties.

Start templates in the Report preview mode

When this function is enabled, the Report preview mode is automatically activated when opening a report template.

Automatically open exported PDF documents

When enabled, the PDF document is automatically opened after the report is saved as PDF.

Allow assigning Macro Datasource to TextBox through inline editor

When enabled, this function allows you to quickly assign the Macro datasource to a Text box by inserting â€œ@â€ and then typing an expression or command name.

Display custom user-set object IDs in the report mode

When enabled, the user object IDs are shown in the report mode.

Recent file list contains, files

Define the number of files contained in the recent file list in the File menu.
