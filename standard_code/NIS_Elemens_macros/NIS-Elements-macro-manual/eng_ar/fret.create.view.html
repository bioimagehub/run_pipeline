Creating FRET View

Any image which contains at least 2 channels can be converted to a FRET image.

  1. Run the Applications > Ratio, Ca2+, FRET > FRET View command. The following window appears:

    Figure 652. 

  2. Select FRET Method (see FRET Method )

  3. Assign one emission type (Da, Dd, Aa, Ad) to one image channel.

  4. Calibrate the module or enter the FRET coefficients by hand. See FRET Calibration.
