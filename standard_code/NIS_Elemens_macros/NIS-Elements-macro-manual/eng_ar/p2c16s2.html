Calo Test

  1. Open your image containing layers to be measured.

  2. Calibrate your image (if not already calibrated) using Calibration > Recalibrate Document.

  3. Run the Applications > Layer Thickness Measurement > Measure Layers command.

  4. Select the CaloTest mode.

  5. Enter the proper Ball Diameter [mm] used for the Calo test.

  6. Click Layer to add a layer.

  7. Name the layer in the Name field.

  8. Select the Color of the layer.

  9. Define a layer using one of the drawing tools. Show Drawing displays a sample Calotest design which can help when defining the layers.

    3pts Circle

    Defines a circular layer by placing three points onto the perimeter of the layer.

    Circle

    Defines a circular layer by clicking into the center of the circular layer and then dragging the mouse outwards to define the perimeter while holding the primary mouse button down.

  10. Check whether this layer will be measured (Measure checkbox).

  11. Repeat steps 6-10 for all layers in your image.

  12. Results of the measurement are displayed in the results table on the right. You can change the Units, export the results to a report ( Report), to MS Excel ( Export to Excel) or you can delete the defined binary layers after clicking Clear Data.
